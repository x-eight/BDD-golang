# BDD-golang - client

## Install buf your pc
```
https://docs.buf.build/installation
```

## Generate grpc client

```sh
$ buf generate "https://github.com/x-eight/BDD-golang.git#branch=proto,subdir=proto"
```

## Start client

```sh
$ go run .
```

## Run tests - comming soon

```sh
$ go test 
```
